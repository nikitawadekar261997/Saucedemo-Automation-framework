Saucedemo Automation

Project Description

This project automates testing for the Saucedemo web application using Selenium WebDriver and TestNG.  
It covers functional and regression testing for workflows such as login, product selection, cart management, and checkout.


Tech Stack
- Language: Java 17  
- Automation Tool: Selenium WebDriver  
- Test Framework: TestNG  
- Build Tool: Maven  
- Browser: Google Chrome 
- Reporting: TestNG Reports
